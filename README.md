# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/brfeitoza/ig-news.git && cd ig-news

# Install Dependencies
$ yarn

# Run Aplication
$ yarn dev
```
Go to http://localhost:3000/ to see the application running.
